About this Extension

This extension automatically neutralizes headlines on nytimes.com by removing emotional language, sensationalism, and partisan framing while preserving all factual information.

It uses Anthropic's Claude AI to rewrite headlines. Configure your API key above to get started. Small blue indicators next to headlines show which ones have been neutralized.
